# Syntecxhub_RestaurantbillingSystem_
Restaurant Billing System The Restaurant Billing System is a console-based Java application designed to automate the billing process in a restaurant. It allows users to enter menu items, specify their price and quantity, and generates an itemized bill including GST.

🔹 Key Features: *Add multiple menu items dynamically *Remove items if entered incorrectly *Display itemized receipt *Automatic GST (5%) calculation *Modular design using classes and methods *User-friendly console interaction 🔹 Technologies Used: *Java *Scanner class for input *Object-Oriented Programming (OOP) concepts
